KSMART 47기 1팀 프로젝트 소개

🌱프로젝트의 목적

도시공간 농지 활용 및 작물 공유 어반팜 플랫폼취미로 텃밭을 가꾸는 사람들! 그리고 관심이 있어 몇 번 키워봤지만 흙만 키우는 사람들이나 아예 시도조차 해보지 못한 사람들을 대상으로 하는 플랫폼입니다. 취미로 키워 주변 사람들에게 상추 한 무더기를 나누어줍니다.

하지만 주변 사람들로는 소비가 불가능한 양의 수확물들. 만약 조금 더 쉽게 나누어 이윤을 창출할 수 있다면 얼마나 좋을까요? 그리고 식물을 잘 기르지 못하는 사람들이 전문가의 재배계획 플랜을 적용하여 쉽게 키울 수 있다면 얼마나 좋을까요?

어반팜 플랫폼은 자신이 키운 작물을 더 쉽게 다른 사람들에게 팔 수 있게하고, 더 쉽게 식물을 키우게 하고, 서로 식물을 키우는 것을 공유하며 지속적인 관심을 유도하여 신규 도시농부들을 유입시키고 지속 가능한 지역사회 도시 농업 활성화를 목표로 합니다.

🎁기대효과

  1. 지역소비를 증가시켜 탄소배출을 감소시키고 도시지역 자연 생태계의 파괴로 인한 사회, 환경, 문화적 문제 해소를 통한 지역사회 환경 개선
  1. 도시농업 진입장벽 완화를 통한 도시농부 양산
  1. 폐기되는 도시농업 농작물을 줄이고, 어반팜 플랫폼을 통해 판매함으로써 잉여 생산물 처리비용을 감소시키고, 이용자의 가계에 도움이 되어 지역사회 구성원의 도시농업 관심 증대
  1. 지역사회 공동체 활성화를 통한 지역주민 삶의 질 향상
  1. 새로운 도시기능의 도입 및 창출을 통한 지역경제 활성화

🔧개발과정 및 프로젝트 기능

개발과정

시스템 구조도 작성, 기능정의, 상세자료 입력, 테이블 스키마, ERD 작성, UI 설계, 네이밍 규칙, 구현, 배포

프로젝트 기능

🍎 회원관리

  • 관리자, 허브크루, 일반회원(농부) 세 가지 권한으로 회원을 관리
  • 로그인, 로그아웃 기능
  • 아이디, 패스워드 찾기 기능

🍏 일반회원의 작물 농사 계획 관리

(농사 계획 관리 - 상세 농사 계획 관리 - 농사 계획 실행 관리)

  1. 나의 재배 계획 보기
    • 사용자가 작성한 작물 재배계획, 상세 작물 재배계획, 작물 재배계획 실행들을 한 눈에 볼 수 있도록 돕는 검색 기능
  1. 작물 재배 계획 관리
    • 사용자의 작물 재배계획의 전체 과정을 관리
    • 현재 작물 재배계획의 단계, 전체 단계를 확인하며, 어느 과정에 있는지 볼 수 있다.
    • 하위 계획으로 세분화된 계획들의 CRUD를 관리
  1. 상세 작물 재배계획 관리
    • 해당하는 작물 재배계획의 상세 재배계획들의 CRUD를 관리
    • 통합계획의 하위 계획으로 세분화한 9개 카테고리 계획들의 CRUD를 관리
  1. 작물 재배 계획 실행 관리
    • 해당하는 작물의 상세 재배계획의 실행을 관리

🍒 작물 상품 등급 자가 검증

  • 해당하는 작물의 상세 재배계획의 실행을 관리

🍊 허브 자판기 관리

  • 허브 내에 있는 사용자가 자신의 작물을 다른 사용자에게 판매하기위해 진열하는 공간을 관리
  • (관리자) : 자판기의 등록, 조회, 수정, 삭제를 하며, 자판기의 상태를 변경할 수 있음
  • (이용자) : 자판기 칸의 이용 등록, 수정, 삭제 할 수 있음.

🍑 플랜픽

  • 플랫폼을 이용하는 사람들이 재배계획을 통해 키우고 있는 작물들의 사진을 포스팅하여 서로 간의 유대감을 형성하고, 작물을 키우는 행위를 통해 소속감을 형성해서 계속해서 작물을 키워 나갈 수 있도록 함
  • 팔로우 기능, 좋아요 기능, 댓글 기능, 이미지 검색 기능

🥝 키트 온라인 판매

  • 허브의 온라인 플랫폼으로 플랫폼에서 제공하는 키트를 구매하고자 하는 사람에게 키트를 판매하는 온라인 상점
  • 상품 카테고리 분류 기능, 상품 검색 기능
  • (관리자) 키트 상품을 등록할 수 있고, 수정, 삭제등을 할 수 있음.
  • (이용자) : 키트 상품을 조회, 상세 조회 할 수 있으며 원하는 상품을 구매, 배송요청 할 수 있음.
